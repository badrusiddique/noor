# Code of Conduct

## Our pledge

We pledge to make participation in the Noor project a respectful, inclusive experience for everyone — regardless of background, identity, experience, or perspective.

## Our standards

Behaviors that build a healthy community:

- Showing empathy and kindness toward others.
- Respecting differing viewpoints and lived experiences.
- Giving and accepting constructive feedback gracefully.
- Taking responsibility for mistakes, apologizing, and learning from them.
- Prioritizing what is best for the community as a whole.

Behaviors that are not welcome here:

- Personal attacks, insulting comments, or sustained disruption.
- Trolling or deliberately inflammatory language.
- Sharing private information about others without permission.
- Conduct that a reasonable person would consider unprofessional.

## Scope

This Code applies in all project spaces — the GitHub repository, issues, pull requests, discussions, code review comments — and any public space where someone is representing the Noor project.

## Enforcement

If you experience or witness behavior that violates this Code, contact the maintainer at **badru.siddique@hudl.com**. All reports will be handled discreetly and reviewed promptly. Privacy of reporters will be respected.

The maintainer may take any action they deem appropriate, including:

- A private warning with an explanation.
- A request for a public apology.
- A temporary or permanent ban from project spaces.
